Detailed Description

Common parts of the MMM family of methods for the electrostatic interaction: MMM1D and ELC. This file contains the code for the polygamma expansions used for the near formulas of MMM1D.

The expansion of the polygamma functions is fairly easy and follows directly from [1]. For details, see [5].

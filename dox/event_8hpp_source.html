23 /** \file
24  * This file contains the hook procedures. These are the ones with names
25  * on_* and are called whenever something is changed in ESPResSo which
26  * might influence other parts. For example, the P3M code has to be
27  * reinitialized whenever the box size changes. The hooking mechanism
28  * allows to keep track of such changes.
29  *
30  * For this mechanism to work, two things have to be fulfilled. If
31  * some part of the code changes some property, it has to call the
32  * corresponding hook, i.e. on_particle_change() if a particle
33  * property has been changed or on_short_range_ia_change(), if a short
34  * ranged interaction has been changed. In turn procedures that
35  * depend on particle properties or the box size, should react to
36  * such changes in the corresponding hook procedure.
37  *
38  * Implementation in event.cpp.
39  */
40

21 /** \file
22  * MMM1D algorithm for long range %Coulomb interactions.
23  * Implementation of the MMM1D method for the calculation of the electrostatic
24  * interaction in one-dimensionally periodic systems. For details on the
25  * method see MMM in general. The MMM1D method works only with the nsquared,
26  * since neither the near nor far formula can be decomposed. However, this
27  * implementation is reasonably fast, so that one can use up to 200 charges
28  * easily in a simulation.
29  */

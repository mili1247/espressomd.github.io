Detailed Description

Molecular dynamics integrator for rotational motion.

A velocity Verlet algorithm using quaternions is implemented to tackle rotational motion. See [34] for the method and [3] for the quaternion components indexing used here. A random torque and a friction term are added to provide the constant NVT conditions. Due to this feature all particles are treated as 3D objects with 3 translational and 3 rotational degrees of freedom if ROTATION is compiled in.

void MMM1D_set_params ( double  switch_rad,
double  maxPWerror 
)

Set parameters for MMM1D.

Most of the parameters can also be tuned automatically. Unlike P3M, this tuning is redone automatically whenever parameters change, but not immediately if you set these parameters.

Parameters
switch_radat which xy-distance the calculation switches from the far to the near formula. If -1, this parameter will be tuned automatically.
maxPWerrorMaximal allowed pairwise error for the potential and force. This error ignores prefactors, i.e. this is for a pure lattice 1/r-sum.

void cuda_mpi_send_forces ( const ParticleRange &  particles,
Utils::Span< float >  host_forces,
Utils::Span< float >  host_torques 
)

Distribute forces to the slaves, and add them to the particles.

Parameters
particlesThe particles for which the forces (and torques) should be added to.
host_forcesThe forces as flat array of size 3 * particles.size(), only relevant on the master.
host_torquesThe torques as flat array of size 3 * particles.size(), this is only touched if ROTATION is active. Only relevant on the master.

This is a collective call.

Detailed Description

Particle cache on the master.

This class implements cached access to all particles in a particle range on the master node. This implementation fetches all particles to the master on first access. Updates of the particle data are triggered automatically on access. The data in the cache is invalidated automatically on_particle_change, and then updated on the next access.

◆ begin()

auto PartCfg::begin ( )
inline

Iterator pointing to the particle with the lowest id.

Returns a random access iterator that traverses the particles in order of ascending id. If the cache is not up-to-date, an update is triggered. This iterator stays valid as long as the cache is valid. Since the cache could be invalidated and updated elsewhere, iterators into the cache should not be stored.

Top-level message.

A message is either updates a property, or a position, or ... New messages can be added here, if they fulfill the type requirements, namely: They either have an integer member id indicating the particle that should be updated with an operator()(const Particle&) that is called with the particle, or a tree of variants with leaves that have such an operator() member.

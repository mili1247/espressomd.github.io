Detailed Description

template<class T>
struct CudaDeviceAllocator< T >

Allocator that uses CUDA to allocate GPU memory.

This allocator uses thrust::malloc and thrust::free to manage device memory. To be able to have static containers for device memory, exceptions in deallocations are suppressed. This is needed because static containers may be destroyed after the CUDA system was teared down, leading to device_free to fail.

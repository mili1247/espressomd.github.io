Detailed Description

template<class T>
class Utils::Bag< T >

Bag of elements.

A bag is a container in which the elements do not have a fixed order. It can be considered an unordered variant of vector, and implements the Container named requirement specified in C++11.

Elements in the container do not have a stable position and removing elements can change the order of the other elements. The loser contract (compared to a vector) allows removing any element in the container in constant time.

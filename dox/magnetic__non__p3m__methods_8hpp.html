Detailed Description

All 3d non-P3M methods to deal with the magnetic dipoles

DAWAANR => Dipolar All With All And No Replica Handling of a system of dipoles where no replicas exist. Assumes minimum image convention for those axis in which the system is periodic.

MDDS => Magnetic Dipoles Direct Sum Calculate dipole-dipole interaction of a periodic system by explicitly summing the dipole-dipole interaction over several copies of the system. Uses spherical summation order.

88 /** Calculate the spherically averaged structure factor.
89  *
90  * Calculates the spherically averaged structure factor of particles of a
91  * given type. The possible wave vectors are given by q = 2PI/L sqrt(nx^2 +
92  * ny^2 + nz^2).
93  * The S(q) is calculated up to a given length measured in 2PI/L (the
94  * recommended order of the wave vector is less than 20).
95  * The data is stored starting with q=1, and contains alternatingly S(q-1) and
96  * the number of wave vectors l with l^2=q. Only if the second number is
97  * nonzero, the first is meaningful. This means the q=1 entries are sf[0]=S(1)
98  * and sf[1]=1. For q=7, there are no possible wave vectors, so
99  * sf[2*(7-1)]=sf[2*(7-1)+1]=0.
100  *
101  * @param[in] partCfg particle collection
102  * @param[in] p_types list with types of particles to be analyzed
103  * @param[in] order the maximum wave vector length in units of 2PI/L
104  * @param[out] wavevectors the scattering vectors q
105  * @param[out] intensities the structure factor S(q)
106  */
107 void calc_structurefactor(PartCfg &partCfg, std::vector<int> const &p_types,
108  int order, std::vector<double> &wavevectors,
109  std::vector<double> &intensities);

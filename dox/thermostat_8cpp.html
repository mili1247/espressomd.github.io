int thermo_switch = THERMO_OFF

Switch determining which thermostat(s) to use.

This is a or'd value of the different possible thermostats (defines: THERMO_OFF, THERMO_LANGEVIN, THERMO_DPD THERMO_NPT_ISO). If it is zero all thermostats are switched off and the temperature is set to zero.

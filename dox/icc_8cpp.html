Detailed Description

Functions to compute the electric field acting on the induced charges, excluding forces other than the electrostatic ones. Detailed information about the ICC* method is included in the corresponding header file icc.hpp.

Calculate the electrostatic forces between source charges (= real charges) and wall charges.

For each electrostatic method, the proper functions for short- and long-range parts are called. Long-range parts are calculated directly, short-range parts need helper functions according to the particle data organisation. This is a modified version of force_calc.
